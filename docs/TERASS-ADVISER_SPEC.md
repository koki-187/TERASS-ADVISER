# TERASS業務アドバイザー Super Agent 仕様書

## 1. システム概要

TERASS業務アドバイザー（Super Agent）は、TERASS社の業務委託不動産エージェント向けに開発された、完全統合型の対話型AIアドバイザーシステムです。

### 目的
- 不動産エージェントの業務効率化
- 報酬計算の自動化と透明性の向上
- Agent Classの判定と昇格支援
- 営業ツールの効果的な活用支援

### 対象ユーザー
- TERASS社の業務委託不動産エージェント
- 新規参入エージェント
- 既存エージェント（全クラス対応）

---

## 2. 主要機能

### 2.1 報酬計算エンジン
高精度な報酬計算システムを提供します。

#### 報酬率体系
- **自己発見案件**: 75%
- **ボーナスステージ**: 90%（年度累計2,000万円達成後、翌月1日から年度末まで自動適用）
- **本部送客**: 40%
- **Terass Offer**: 55%

#### 計算基準
- 税抜仲介手数料をベースに計算
- 自動判定によるボーナス適用
- 複数案件の合算対応

#### 機能詳細
```python
# 基本報酬計算
基本報酬 = 税抜仲介手数料 × 報酬率

# ボーナスステージ判定
if 年度累計 >= 20,000,000:
    報酬率 = 90%  # 翌月1日から適用
```

### 2.2 Agent Class 判定システム
エージェントのクラス判定と昇格支援を行います。

#### 8クラス体系
1. **Premier Agent**
2. **Executive Agent**
3. **Master Agent**
4. **Senior Agent**
5. **Professional Agent**
6. **Lead Agent**
7. **Standard Agent**
8. **Junior Agent**

#### 判定基準
- 首都圏/地方別基準の自動適用
- 半期売上基準
- 累計成約件数
- 昇格までの差額・道のりの算出

#### 特典体系
各クラスごとに以下の特典が付与されます：
- ブランディング支援
- 優先サポート
- 専用ツールへのアクセス
- 特別報酬プログラム

### 2.3 TERASS Picks 最適化
複数ポータルの統合検索と最適化提案を行います。

#### 対応ポータル
- SUUMO
- at HOME
- REINS（東日本・中部・西日本・近畿）

#### 主な機能
- 統合検索機能
- ユーザーリアクション分析
- Terass Offerとの比較提案
- 自己発見案件として報酬75-90%の適用

### 2.4 住宅ローン最適提案
107行の金融機関に対応した住宅ローン提案システム。

#### 対応金利範囲
- 最低金利: 0.520%
- 最高金利: 3.242%
- 変動金利/固定金利の自動判定

#### 提案機能
- 顧客条件にマッチした銀行の優先提案
- 金利比較シミュレーション
- 審査通過可能性の判定

### 2.5 Terass Cloud 連携
業務管理システムとの完全連携。

#### 対応機能
- 商談管理
- 申請管理
- 契約管理
- ダッシュボード表示
- 帳票自動生成

## 3. 技術仕様

### 3.1 アーキテクチャ
3層のティア構造で情報を管理します。

#### Tier 1 — 基本情報
- Notion 12セクションの完全同期
- リアルタイムアクセス
- 会社理念・基本方針の参照

#### Tier 2 — 数値・基準
- 報酬率計算エンジン
- Agent Class判定アルゴリズム
- 住宅ローン金利比較ロジック

#### Tier 3 — 実践ツール
- TERASS Picks活用ガイド
- Terass Cloud業務フロー
- Loan Checker提案ロジック

### 3.2 データソース
- **Notion**: 業務マニュアル、報酬体系（12セクション）
- **報酬マニュアル**: 全18条
- **Agent Class基準**: 8クラス×地域別
- **金融機関データ**: 107行の金利情報
- **不動産ポータル**: SUUMO/at HOME/REINS

### 3.3 更新頻度
- **業務マニュアル**: 毎日自動チェック
- **住宅ローン金利**: 外部データから自動更新
- **Agent Class基準**: 四半期ごとに確認
- **報酬体系**: 年度更新

| Lead | 500万円以上 | 10件以上 |
| Standard | 300万円以上 | 6件以上 |
| Junior | - | - |

### B. 報酬計算式詳細
```
基本報酬 = 税抜仲介手数料 × 報酬率

報酬率の判定:
- 自己発見: 75%
- ボーナス（年度累計2,000万円以上達成翌月1日〜年度末）: 90%
- 本部送客: 40%
- Terass Offer: 55%

ボーナスステージ適用条件:
- 年度累計売上が2,000万円に達した月の翌月1日から年度末まで
- 自己発見案件に適用（90%）
```

### C. 用語集
- **自己発見案件**: エージェント自身が獲得した案件
- **本部送客**: TERASS本部から紹介された案件
- **Terass Offer**: TERASS社提供の特別案件
- **TERASS Picks**: 複数ポータル統合検索システム
- **Terass Cloud**: 業務管理クラウドシステム
- **Agent Class**: エージェントの実績に基づくランク制度
