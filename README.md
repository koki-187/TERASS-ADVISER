## 🎯 システム概要
完全に統合された対話型 AI アドバイザーとして、TERASS 社の業務委託エージェント向けに以下の全機能を実装しています。  
2025年10月時点の公式資料に基づく情報を元に動作します。推測や不確実な情報は含みません。

主な提供機能：
- 報酬計算エンジン（高精度）
- Agent Class（エージェントクラス）判定システム
- TERASS Picks（ポータル統合）最適化アドバイザー
- 住宅ローン最適提案（107銀行対応）
- Terass Cloud を含む業務連携（申請・契約・帳票）
- コンプライアンス検証、トラブルシューティング

---

## 🧠 コア機能詳細

### 💰 報酬計算エンジン
- 基本報酬：自己発見案件 75%
- ボーナスステージ：年度累計 2,000 万円達成後は翌月1日～年度末まで 90% 自動適用
- 本部送客：40%
- Terass Offer：55%
- 税抜仲介手数料ベースで計算
- 自動判定によるボーナス適用・複数案件合算対応

### 📊 Agent Class 判定システム
- 8 クラス対応（Premier ～ Junior）
- 首都圏 / 地方別基準の自動適用
- 昇格までの差額・道のりを算出して提示
- 各クラスごとの特典詳細を表示

### 🏠 TERASS Picks 最適化
- SUUMO / at HOME / REINS の主要 3 大ポータルを統合検索
- ユーザー反応（リアクション機能）を活用した提案アドバイス
- Terass Offer との比較提案機能
- TERASS Picks を「自己発見案件」として扱い、報酬 75–90% の判定ロジックを適用

### 💳 住宅ローン最適提案
- 107 行・金融機関の金利データに対応（自動更新前提）
- 金利帯：0.520% ～ 3.242%
- 変動 / 固定金利の自動判定
- 顧客条件にマッチした銀行の優先提案

### 📋 Terass Cloud 連携
- 商談管理、申請管理、契約管理に対応
- ダッシュボードによる状況把握
- 帳票の自動生成対応
